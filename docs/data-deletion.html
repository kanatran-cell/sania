Politica de Eliminacion de Datos

SanIA - Tu detective de alimentos

SanIA no almacena datos personales. No hay datos que eliminar.

Datos temporales

SanIA solo almacena datos temporales durante una sesion de uso:

  • Fotografias de productos (en cache temporal del dispositivo)
  • Resultados de comparacion (en memoria temporal)

Estos datos se eliminan automaticamente al:

  • Iniciar una nueva comparacion (boton "Borrar todo")
  • Cerrar la aplicacion
  • Desinstalar la aplicacion

Eliminacion completa

Para eliminar completamente todos los datos asociados a SanIA:

  1. Abre Ajustes de tu dispositivo Android
  2. Ve a Aplicaciones > SanIA
  3. Toca Almacenamiento > Borrar datos y Borrar cache
  4. Opcionalmente, desinstala la aplicacion

Sin datos en servidores

SanIA no almacena ninguna informacion en servidores externos. No existe una base de datos remota con informacion de usuarios. Por lo tanto, no hay datos en la nube que necesiten ser eliminados.
